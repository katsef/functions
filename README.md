### validate_russian_phone_number

**Валидация строки содержащую телефонный номер.**

------

#### Описание

```php
string validate_russian_phone_number  (string $tel)
```


Функция принимает на входе строку с телефонным номером в произвольном формате, а возвращает либо телефонный номер в формате `+7xxxxxxxxxx,` либо `false`

#### Список параметров

***$tel*** - строка с телефонным номером любого формата, например +7 (90X) XXX-XX-XX

#### Возвращаемые значения

Возвращает строку (string) в формате `+7xxxxxxxxxx,` либо `false`

### normalise_phone

**Отбрасывает первый символ "+" в строке номера телефона, если такой присутствует.**

#### Описание

```php
string normalise_phone (string $phone)
```

#### Список параметров

***$phone*** - строка с валидным телефонным номером

#### Возвращаемые значения

Строка (string) содержащая номер телефона без "+"
